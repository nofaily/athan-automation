# Contributing to Athan Automation

First off, thank you for considering contributing to Athan Automation! It's people like you that make this project better for the Muslim community worldwide.

## How Can I Contribute?

### Reporting Bugs

Before creating bug reports, please check the existing issues to avoid duplicates. When you create a bug report, include as many details as possible:

- **Use a clear and descriptive title**
- **Describe the exact steps to reproduce the problem**
- **Provide specific examples** to demonstrate the steps
- **Describe the behavior you observed** and what you expected to see
- **Include logs** from `/var/log/athan-automation/`
- **Specify your environment**: OS, Python version, Chromecast model

### Suggesting Enhancements

Enhancement suggestions are tracked as GitHub issues. When creating an enhancement suggestion, include:

- **Use a clear and descriptive title**
- **Provide a step-by-step description** of the suggested enhancement
- **Provide specific examples** to demonstrate the steps
- **Describe the current behavior** and the behavior you want to see
- **Explain why this enhancement would be useful**

### Pull Requests

1. Fork the repo and create your branch from `main`
2. If you've added code that should be tested, add tests
3. Ensure your code follows the existing style
4. Update documentation as needed
5. Write a clear commit message

## Code Style Guidelines

- Follow PEP 8 style guide for Python code
- Use meaningful variable and function names
- Add comments for complex logic
- Keep functions focused and concise
- Use type hints where appropriate

## Testing

Before submitting a pull request:

1. Test your changes on actual Chromecast hardware
2. Verify all five prayer times work correctly
3. Test Ramadan mode if applicable
4. Check that configuration hot-reload works
5. Ensure logs are being written correctly

## Documentation

- Update README.md if you change functionality
- Add docstrings to new functions
- Update configuration examples if needed

## Community

- Be respectful and considerate
- Welcome newcomers and encourage diverse perspectives
- Focus on what is best for the community
- Show empathy towards other community members

## Questions?

Feel free to open an issue with your question or reach out to the maintainers.
